    31 - Coin Sums

    how many ways can you make $2 using the following coins: 1c, 2c, 5c, 10c, 20c, 50c, $1, $2

    initial thoughts, i'll define p(n) to be the number of ways you can reach n using the set of coins.
    this means p(0) = p(1) = 1.
    for every i, p(i+1) >= p(i), you can attach 1p to every way in p(i).
    what about other coins? it is not as simple as p(n) = Σ꜀ p(n-c), for c in coins as i first thought because this double counts many solutions.
    this when i figured out all i had to do to stop double counting is make sure the coins picked in descending order.
    this makes so if a coin c is picked first, then no coin greater then c can be picked until the target is reached.
